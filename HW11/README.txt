RestApi Project

Цей проект є прикладом REST API, побудованим за допомогою **FastAPI** та **SQLAlchemy** для роботи з базою даних. Проект реалізує функціонал для зберігання та отримання контактів, а також для отримання майбутніх днів народжень.

Встановлення

1. Клонуйте репозиторій:
   ```bash
   git clone <url-вашого-репозиторію>
   cd RestApi
   ```

2. Встановіть залежності за допомогою Poetry:
   ```bash
   poetry install
   ```

3. Активуйте віртуальне середовище:
   ```bash
   poetry shell
   ```

4. Для запуску сервера використовуйте команду:
   ```bash
   poetry run uvicorn main:app --reload
   ```

   Сервер буде доступний за адресою [http://127.0.0.1:8000](http://127.0.0.1:8000).

Можливості API

1. Отримати список всіх контактів:
   - **GET** `/contacts/`
   - Приклад запиту:
     ```bash
     curl -X 'GET' 'http://127.0.0.1:8000/contacts/'
     ```

2. Пошук контактів за запитом:
   - **GET** `/contacts/search/`
   - Параметр запиту: `query`
   - Приклад запиту:
     ```bash
     curl -X 'GET' 'http://127.0.0.1:8000/contacts/search/?query=1'
     ```

3. Отримати список контактів з майбутніми днями народженнями:
   - **GET** `/contacts/birthdays/`
   - Параметр запиту: `days_ahead` (за замовчуванням 7)
   - Приклад запиту:
     ```bash
     curl -X 'GET' 'http://127.0.0.1:8000/contacts/birthdays/?days_ahead=7'
     ```

Розробка

- Для локальної розробки можна використовувати команду `uvicorn` з параметром `--reload`, що дозволяє автоматично перезавантажувати сервер при зміні коду:
  ```bash
  poetry run uvicorn main:app --reload
  ```

Технології

- **FastAPI** — для створення API.
- **SQLAlchemy** — для взаємодії з базою даних.
- **PostgreSQL** — база даних (або інша, в залежності від конфігурації).
- **Uvicorn** — сервер для запуску FastAPI.
- **Poetry** — для управління залежностями.
