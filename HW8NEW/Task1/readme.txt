# Проект: Цитати

## Опис

Цей проект реалізує систему для зберігання та пошуку цитат, використовуючи MongoDB як базу даних,
Redis для кешування та Python як основну мову програмування. Проект дозволяє користувачам шукати
цитати за іменами авторів або тегами, а також підтримує можливість кешування результатів для
підвищення продуктивності.

## Функціональність

1. **Пошук цитат**:
    - **За іменем автора**: Користувачі можуть вводити команду у форматі `name:<ім'я автора>`, щоб отримати цитати, написані конкретним автором.
    - **За тегом**: Команда `tag:<тег>` дозволяє шукати цитати за вказаним тегом.
    - **За кількома тегами**: Команда `tags:<тег1,tег2,...>` дозволяє шукати цитати за кількома тегами одночасно.

2. **Кешування**:
    - Результати запитів кешуються в Redis для пришвидшення повторних запитів.

## Налаштування

1. **Встановлення залежностей**:
    Використовуйте [Poetry](https://python-poetry.org/) для управління залежностями:
    ```bash
    poetry install
    ```

2. **Налаштування MongoDB**:
    - Створіть обліковий запис MongoDB на [MongoDB Atlas](https://www.mongodb.com/cloud/atlas).
    - Отримайте `MONGO_USERNAME`, `MONGO_PASSWORD` та `MONGO_DB` для підключення до бази даних.
    - Або використовуйте мою базу данних.
3. **Налаштування Redis**:
    - Для локального запуску Redis, можете скористатись Docker:
    ```bash
    docker run -d -p 6379:6379 redis
    ```
    - Або запустіть docker-compose.yaml

4. **Завантаження змінних середовища**:
    -В файлі `.env` у кореневій директорії проекту є такі змінні:
    ```
    MONGO_USERNAME=ваше_ім'я_користувача
    MONGO_PASSWORD=ваш_пароль
    MONGO_DB=ваша_база_даних
    ```
