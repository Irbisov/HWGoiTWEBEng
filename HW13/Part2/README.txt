
# Django Project - Скидання паролю

Цей проект є базовим веб-додатком на Django, що включає механізм скидання паролю для зареєстрованих користувачів, а також налаштування поштових сервісів для надсилання інструкцій щодо скидання паролю. Проект також містить функціональність для реєстрації та автентифікації користувачів.

## Технології

- **Django**: для створення веб-додатку.
- **PostgreSQL**: для зберігання даних користувачів та інших даних.
- **SMTP сервер**: для надсилання листів з інструкціями для скидання паролю.

## Створення та налаштування середовища

1. **Клонування репозиторію**:

   Спочатку клонуватимемо репозиторій на ваш комп'ютер:
   ```bash
   git clone https://github.com/your-username/django-password-reset.git
   cd django-password-reset
   ```

2. **Віртуальне середовище**:

   Створіть і активуйте віртуальне середовище:
   ```bash
   python -m venv venv
   source venv/bin/activate  # для Windows використовуйте `venv\Scriptsctivate`
   ```

3. **Встановлення залежностей**:

   Встановіть необхідні пакети:
   ```bash
   pip install -r requirements.txt
   ```

4. **Налаштування бази даних**:

   Потрібно налаштувати PostgreSQL або іншу базу даних. В файлі `settings.py` вкажіть параметри підключення до вашої бази даних:

   ```python
   DATABASES = {
       'default': {
           'ENGINE': 'django.db.backends.postgresql',
           'NAME': 'dbhw101',
           'USER': 'mr.green',
           'PASSWORD': 'your-password',
           'HOST': 'localhost',
           'PORT': '5432',
       }
   }
   ```

5. **Міграція бази даних**:

   Застосуйте міграції для створення таблиць:
   ```bash
   python manage.py migrate
   ```

6. **Налаштування пошти**:

   У файлі `settings.py` додайте налаштування для пошти:

   ```python
   EMAIL_BACKEND = 'django.core.mail.backends.smtp.EmailBackend'
   EMAIL_HOST = 'smtp.gmail.com'
   EMAIL_PORT = 587
   EMAIL_USE_TLS = True
   EMAIL_HOST_USER = 'your-email@gmail.com'
   EMAIL_HOST_PASSWORD = 'your-email-password'
   ```

   **Примітка**: Для тестування ви можете використовувати консольний поштовий бекенд:

   ```python
   EMAIL_BACKEND = 'django.core.mail.backends.console.EmailBackend'
   ```

7. **Створення суперкористувача**:

   Створіть суперкористувача для доступу до адміністративної панелі Django:
   ```bash
   python manage.py createsuperuser
   ```

8. **Запуск сервера**:

   Тепер ви готові запустити сервер:
   ```bash
   python manage.py runserver
   ```

   Доступ до вашого проекту буде на `http://127.0.0.1:8000`.

## Особливості

- **Скидання паролю**:
  - Користувачі можуть скинути свій пароль, отримавши посилання для відновлення через email.
  - Процес складається з 4 етапів:
    1. Користувач вводить свою email-адресу.
    2. Відправляються інструкції на email.
    3. Користувач переходить по посиланню та вводить новий пароль.
    4. Повідомлення про завершення скидання паролю.

- **Шаблони**:
  - Проект включає стандартні шаблони для реєстрації користувачів та скидання паролю.

## Відомі проблеми

1. Якщо email не відправляється, перевірте налаштування SMTP сервера.
2. Під час локального тестування використовуйте `console.EmailBackend` для перегляду листів у консолі.
